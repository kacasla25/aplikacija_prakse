# Aplikacija za stručne prakse

Web aplikacija namenjena povezivanju kandidata koji traže stručnu praksu i kompanija koje objavljuju oglase za praksu.

Projekat je napravljen korišćenjem PHP-a i MySQL baze podataka.

## Funkcionalnosti

- Registracija novih korisnika
- Prijava postojećih korisnika
- Izbor tipa korisnika:
  - kandidat
  - firma
- Različito preusmeravanje korisnika nakon prijave
- Čuvanje korisničkih podataka u MySQL bazi
- Objavljivanje oglasa za stručne prakse
- Prijavljivanje kandidata na oglase
- Praćenje statusa prijave
- Evidencija procesa selekcije
- Paketi za objavljivanje oglasa

## Tipovi korisnika

### Kandidat

Kandidat može da pregleda oglase za praksu i prijavi se na izabrani oglas.

### Firma

Firma može da kreira svoj profil, objavljuje oglase za praksu i upravlja prijavama kandidata.
